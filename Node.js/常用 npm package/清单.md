* [minimist](https://www.npmjs.com/package/minimist)

    格式化node命令中的参数，将其变成对象
    ```js
    var argv = require("minimist")(process.argv.slice(2));
    console.log(argv);
    ```
    ```powershell
    $ node example/parse.js -x 3 -y 4 -n5 -abc --beep=boop foo bar baz
    { _: [ 'foo', 'bar', 'baz' ],
      x: 3,
      y: 4,
      n: 5,
      a: true,
      b: true,
      c: true,
      beep: 'boop' }
    ```
